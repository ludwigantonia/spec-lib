## Figures

### Figure 3 — Taxonomic Composition of the Spectral Library
**[View interactive figure](https://ludwigantonia.github.io/spec-lib/sunburst_arctic_tundra.html)**

Interactive sunburst diagram showing the taxonomic composition of the Arctic 
tundra spectral library across three functional groups (vascular plants, 
bryophytes, lichens), organized by family and genus. Circle segments are 
proportional to the number of species represented in the current version of the library.
(Based on 5,739 records that included the relevant taxonomic
information.)
Built with [plotly](https://plotly.com/r/) in R.

## Data

The full spectral library is currently not openly available in its harmonised operational version as we do not 
own the copyright or ownership of the individual datasets. 
All datasets are cited in their original version in Section 6.1 of the manuscript (Appendix - List of datasets).
